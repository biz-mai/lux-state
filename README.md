# LuxEstate PH

Premium Real Estate & Investment platform for Metro Manila.

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Vercel Serverless Functions (contact form)
- Nodemailer (Gmail SMTP)

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values before running locally.

## Deploy

Push to GitHub and import on [Vercel](https://vercel.com). Add env vars in Project Settings.
